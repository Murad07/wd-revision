Markup Tag Structure:

  • HTML - it's a HyperText Markup Language
  • There are different types of Tag as -
    • <p></p>- tag for pragraph
    • <b></b> - tag for bold text
    • <input type="text"> - tag for input box
    • <strong></strong> - same as b tag
    • Heading tags - h1, h2 to h5
    • <a href=""></a> - anchor tag
    • <img src="" alt=""> - image tag
  • Tag attribute -
    • href - a attribute of anchor tag
    • src - a attribute of img tag
    • alt and more

How to display image

  • <img src="my-image-url" alt="my-text" />
  • Use img tag to display image
